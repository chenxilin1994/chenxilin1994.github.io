三、评论区功能（使用Giscus） ​

  1. 在config.mts中添加：
typescript
export default defineConfig({
  //...原有配置
  
  markdown: {
    //...原有配置
    config: (md) => {
      md.use(require('vitepress-plugin-comment/giscus'), {
        repo: 'yourgithubname/yourrepo', // 你的github仓库
        repoId: 'YOUR_REPO_ID',          // 在Giscus设置页面获取
        category: 'Announcements',       // Discussion分类
        categoryId: 'YOUR_CATEGORY_ID',  // 分类ID
        mapping: 'pathname',             // 按页面路径关联评论
        reactionsEnabled: true,          // 启用表情反应
        emitMetadata: false,
        theme: 'preferred_color_scheme'  // 自动切换主题
      })
    }
  }
})

四、RSS订阅配置 ​

  1. 创建docs/.vitepress/rss.ts：

import { RssOptions } from 'vitepress-plugin-rss'

export const rssOptions: RssOptions = {
  title: '智学笔记',
  baseUrl: 'https://yourdomain.com',
  description: '编程与人工智能技术博客',
  language: 'zh-CN',
  copyright: 'Copyright © 2024 YourName',
  filter: (page) => /^\/blog\//.test(page.url), // 过滤需要生成RSS的路径
  filename: 'feed.xml',
  customElements: [
    { 
      name: 'author',
      value: 'YourName <your@email.com>' 
    }
  ],
  async: true
}

注意事项 ​

  1. Algolia申请需要满足以下条件：

    • 网站必须公开可访问
    • 有完整的技术文档内容
    • 非商业用途可免费使用

  2. Giscus要求：

    • GitHub仓库必须公开
    • Discussions功能已启用
    • Giscus应用已授权

  3. RSS生成：

    • 每次构建会自动生成feed.xml
    • 访问地址为：https://yourdomain.com/feed.xml

  4. 安全建议：

    • 所有API密钥应通过环境变量注入
    • 不要将敏感信息提交到版本库

建议按顺序逐步添加这些功能，添加完成后运行 npm run build 检查构建结果。
